# Banking Management System

## Description
This is a Java-based Banking Management System developed as a 2nd semester OOP project.

## Features
- Account creation (Current, Saving, Islamic)
- Deposit and Withdrawal
- Balance Inquiry
- Loan Management
- File handling for data storage
- JavaFX GUI

## Technologies Used
- Java
- JavaFX
- Object-Oriented Programming (OOP)
